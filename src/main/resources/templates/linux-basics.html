Lecture Notes

Unix is a very powerfull operating system. It was originally developed in 1991 by Linus Torvalds. it is opensource. Being opensource one is able to correct it or add certain features to the source code. There are different flavors of LINUX os:

  1. Redhat
  2. fedora
  3. Ubuntu
  4. Mint

Linux has all the typical component of an operating system. for example The kernel which is found in boot/vmlinuz and the source files are found in the /usr/src/linux
The Linux operating system can be commanded in one of two ways. One is the GNOME which is the graphical user interface for linux.
