An Operating system is a software that manages computers hardware. It also provides a basis for applications to act as intermediary to the computer user and the computer hardware.
Operating systems are everywhere:

  • cars
  • In the Internet of things
  • On smartphones
  • On Cloud computers.

Organization of Computer hardware

Computer hardware is made up of:

  • CPU
  • Memory
  • IO Devices
  • Storage

A fundamental responsibility of an operating system is to allocate these resources to programs.

Components of a computer system

A computer system can be roughly divided into four components

  1. User
  2. Application Programs
  3. Operating System
  4. Computer Hardware

The operating system is the one program that is running all the time on the computer and is usually called the kernel

Types of Operating Systems

There are 6 types of operating systems

1. Batch Operating Systems


The Batch operating System is appropriate for lengthy, time consuming tasks. These systems are usually payroll systems, data entry and bank statements.
Advantages
  1. Many users can share a batch systems
  2. It becomes possible to handle large workloads
  3. It's easy to estimate how long a task will take to be completed.
Disadvantages
  1. batch operating systems are challenging to debug
  2. Any failure of a batch operating system create a backlog
  3. It may be costly to install and maintain a good batch operating system

2. Time Sharing or Multitasking operating system


Works by allocating time to a particular task and switching between tasks frequently.
Advantages
  1. There is a quick response during task performance.
  2. It minimizes the idle time of the processor.
  3. All tasks get an equal chance of being completed.
  4. IT reduces the chance of software duplication
Disadvantages
  1. The user's data security might be a problem.
  2. System failure might lead to widespread failures.
  3. Problems in data communication may rise.
  4. The integrity of the user programs is not assured.

3. Distributed OS


This system is based on autonomous but interconnected computers communicating with each other via communication lines or a shared network.
Advantages
  1. They allow remote working.
  2. They allow faster exchange of data between users.
  3. Failure in one system may not cause failure of the whole system.
  4. They reduces delays in data processing.
  5. They minimize the load on the host computer.
  6. They enhance scalability
Disadvantages
  1. If the primary network fails the whole network fails
  2. They're expensive to maintain and require a high level of expertise.

4. Network Operating System


Network operating systems are installed on a server providing users with the capability to manage data, user groups and applications.
Advantages
  1. Centralised server provide high stability.
  2. Security issues are easy to handle through the server.
  3. It's easy to upgrade and integrate new technologies.
  4. Remote access to server is poss`ible.
Disadvantages
  1. They require regular updates and maintenance.
  2. Server are expensive to buy and maintain.
  3. The users reliance to the server might be detrimental to the workflows.

5. Real Time Operating System


Real-time operating systems provide support to realtime systems that require observance of strict time requirements.
They are typically used for tasks such as scientific experiments, medical imaging, robotics and air traffic control operations.
Advantages
  1. The use devices and system maximally hence more output.
  2. They allow fast shifting from one task to another.
  3. The focus is on current task and less focus on the queue.
  4. They can be used in embed systems.
  5. They are meticulously programmed hence free of error.
  6. They allow easy allocation of memory
Disadvantages
  1. They have a low capacity for running tasks semultaniously

6. Mobile Operating Systems


Mobile operating systems such as Android OS, Apple and Windows mobile OS run exclusively on small devices such as smartphones, tablets and wearables. • The system combines the features of a personal computer with additional features useful for a handheld device. • Mobile operating systems start when a device is powered on to provide access to installed applications • Mobile operating systems also manage wireless network connectivity
Advantages They are easy for users to learn and use.
Disadvantages They are sommetimes heavy and may drain battery and require frequent charging
